
하나의 주기억장치에 2개이상의 프로그램을 기억시키고 하나의 CPU와 대화하듯 => 다중 프로그래밍 시스템 

컴퓨터가 사용자 프로그램 번갈어 가며 처리 / 각 사용자에게 독립된 컴퓨터를 사용하는 느낌 => 시분할 

일정량 또는 일정 기간 동안 데이터를 모아서 한꺼번에 처리하는 방식 => 일괄 처리 시스템 

여러개의 CPU와 하나의 주기억장치를 이용하여 하나 또는 여러개의 프로그램을 동시에 처리하는 방식 => 다중 처리 시스템 


일괄 처리 시스템: 일정량 또는 일정 기간 동안 데이터를 모아서 한꺼번에 처리하는 방식.  / 급여 계산 지불 계산 연말 결산 등의 업무에 사용함

실시간 처리 시스템: 데이터 발생 즉시 / 처리 요구가 있는 즉시 처리해서 결과 산출 / 우주선 운행/ 레이더 추적기, 핵물리학, 실험 및 데이터 수집 

모아서 하냐 한번에 하냐 

다중 프로그래밍 : 하나의 CPU와 주기억 장치를 이용하여 여러개 동시 처리 / 하나 의 주기억장치에 2개이상 기억 동시대화 

시분할 시스템: 라운드로빈 방식이라고도함. 여러명의 사용자가 사용하는 시스템에서 컴퓨터가 사용자들의 프로그램을 번갈아가며 처리해줌으로써 각 독립된 컴퓨터를 사용하는 느낌을 주는 방식 .

다중처리 : 여러 개의 CPU와 하나의 주기억장치를 이용하여 여러 개의 프로그램을 동시에 처리하는 방식 하나 CPU 가 고장나도 다른 CPU를 이용하여 업무를 처리할 수 있으므로 시스템의 신뢰성과 안정성이 높음. 

다중 모드 처리 : 일괄 처리 시스템, 시분할 시스템, 다중 처리 시스템, 실시간 처리 시스템을 한  시스템에서 제공하는 방식. 

분산 처리 시스템 : 여러 개의 컴퓨터를 통신 회선으로 연결하여 하나의 작업을 처리하는 방식. / 각 단말장치나 컴퓨터 시스템은 고유의 운영체제와 CPU 메모리를 가지고있음.

1개의 정식 IP 주소에 대량의 가상 사설 IP주소를 할당 및 연결하는 방식. 
IP 부족 문제를 해결하기 위한 방법 중의 하나.
1개의 IP주소를 사용해서 외부에 접속할 수 있는 노드는 어느 시점에서 1개만으로 제한되는 문제가 있으나, IP마스커레이드를 이용하여 해결 

사설 IP를 공인 IP로 바꿔주는게 NAT 

1개의 정식 IP주소 => 공인 IP 주소 여기에 가상 사설 IP주소 연결하는 방식 

IP 마스커레이드 리눅스의 NAT기능으로 리눅스 서버를 통해 다른 네트워크와 접속할수 있도록 해줌.
