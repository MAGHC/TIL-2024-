분산 처리 시스템 : 여러 개의 컴퓨터를 통신 회선으로 연결하여 하나의 작업을 처리하는 방식. / 각 단말장치나 컴퓨터 시스템은 고유의 운영체제와 CPU 메모리를 가지고있음.
