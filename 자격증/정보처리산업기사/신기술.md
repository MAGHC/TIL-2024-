
# SW 

인공지능 : 인간의 두뇌와 같이 스스로 추론, 학습, 판단 등 인간지능적인 작업을 수행 

뉴럴링크 : 머스크가 설립한 회사 / 신경 레이스로 전극을 뇌에 이식해서 생각을 업로드하고 다운로드하는것을 목표로 삼고있음

딥러닝: 인간의 두뇌를 모델로 만들어진 인공 신경망을 기반으로 하는 학습 기술 

전문가 시스템: 의료 진단 등과 같은 특정 분야의 전문가가 수행하는 고도의 업무를 지원하는 응용프로그램

증강 현실: 혼합현실(MR)이라고도 부름 

블록 체인: P2P 네트워크를 이용해서 온라인 금융 거래 정보를 온라인 네트워크 참여자(Peer)의 디지털 장비에 분산 저장하는 기술

분산 원장 기술: 중앙 관리자가 중앙 데이터 저장소가 존재안하고 P2P망 내의 참여자들에게 모든 거래 목록이 분산 저장되어 거래가 발생할 때마다 지속적으로 갱신되는 디지털 원장을 의미 

해시 : 메시지를 고정된 길이의 값이나 키로 변환하는 것 (정보가 위조됬냐 아니냐를 확인하기 위함)

양자 암호키 분배: 양자 통신을 위해 비밀키를 분배하여 관리하는 기술 / 두 시스템이 암호 알고리즘 동작을 위한 비밀키를 안전하게 공유하기 위해 양자 암호키 분배 시스템을 설치하여 운용하는 방식으로 활용 

프라이버시 강화 기술: 개인정보 침해 위험을 관리하기 위한 핵심 기술 

공통 평가 기준 : ISO15408 표준으로 채택된 정보보호 제품평가 기준/ 정보화 순기능 역할을 보장하기 위해 정보화 제품의 정보 보호 기능과 이에 대한 사용 환경 등급을 정한 기준 

그레이 웨어 : 회색 / 소프트웨어 제공하는 입장에서 유용할거같다 주장하는데 사용자 입장에서 악의적일수있는거 

메시업: 각종 컨텐츠 , 서비스를 융합해서 새로운것을 만들어내는것 

리치 인터넷 앱: 플래시 애니 + 웹 서버 앱 기술 통합하여 인터렉티브한 웹을 제공하는 신개념 플래시 웹 

시맨틱 웹: 컴터가 잘 읽을수있도록 하는 지능형 웹 

증발품: 판다고했는데 실제로는 안파는 소프트웨어 

오픈 그리드 서비스 아키텍쳐: 앱 공유를 위한 웹서비스를 그리드 상에서 제공하기 위해 만든 개방형 표준 

서비스 지향 아키텍쳐: SOA 기반 앱 구성 계층

복잡 이벤트 처리:  실시간으로 발생하는 많은 사건들 중 의미가 있는 것만을 추출할 수 있도록 조건을 정의하는 데이터 처리 방법 

디지털 트윈: 현실속의 사물을 소프트웨어로 가상화한 모델 / 자산 최적화 , 돌발 ㅅ사고 최소화 등의 모든 과정의 효율성을 향상 시킬 수 있음
